SiD-DiT Tutorial

Quick start guide for running distilled flow-matching T2I models.

Back to Main Page

1. Installation

pip install diffusers transformers accelerate safetensors
pip install -U xformers # optional, for faster attention

2. Basic usage with SD3 / SD3.5

from diffusers import StableDiffusion3Pipeline
import torch

pipe = StableDiffusion3Pipeline.from_pretrained(
    "YGu1998/SiD-Flow-SD3-medium",
    torch_dtype=torch.float16
).to("cuda")

prompt = "a cinematic photo of a lone astronaut walking through a neon-lit city at night"
image = pipe(prompt, num_inference_steps=4).images[0]
image.save("sid_dit_sd3_example.png")

3. Using FLUX.1-Dev distilled models

from diffusers import FluxPipeline
import torch

pipe = FluxPipeline.from_pretrained(
    "YGu1998/SiD-Flow-FLUX-1024-res",
    torch_dtype=torch.float16
).to("cuda")

prompt = "a wide-angle landscape of mountains at sunrise in painterly style"
image = pipe(prompt, num_inference_steps=4).images[0]
image.save("sid_dit_flux_example.png")

4. Reproducible prompts for qualitative comparison

We recommend using a fixed prompt set for comparing teachers vs distilled models. For example:

prompts = [
    "a studio portrait of an elderly woman smiling, soft window light, 85mm lens",
    "a bustling night street scene in Tokyo with neon signs, rain reflections, cinematic",
    "a watercolor painting of a small cottage in a misty forest, soft pastel colors",
    "a dramatic landscape of snow-capped mountains under a purple sunset sky",
]

You can reuse these prompts across SANA, SD3, SD3.5 and FLUX variants to reproduce the figures on the main page.
